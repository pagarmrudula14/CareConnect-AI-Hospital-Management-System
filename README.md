# AI Hospital Management System

This is a Django-based AI Hospital Management System developed as a college project.

## 🔹 Features
- User Signup & Login
- AI Symptom Checker
- Hospital List Display
- Doctor List Based on Hospital
- Appointment Booking System

## 🔹 Technology Used
- Python
- Django
- HTML, CSS, Bootstrap
- SQLite Database
- Git & GitHub

## 🔹 Project Flow
1. User logs in or signs up
2. User enters symptoms
3. System predicts disease
4. Hospitals are displayed
5. Doctors are listed based on hospital
6. Appointment booking is done

# CareConnect-AI-Hospital-Management-System
## Overview
AI Hospital Management System is a web-based healthcare platform designed to simplify hospital operations and improve patient experience through smart digital solutions.

The system includes:
- AI-based Symptom Checker
- Doctor Appointment Booking
- Patient Record Management
- Secure Authentication System

## Tech Stack
- HTML
- CSS
- JavaScript
- Django
- Python
- SQLite/MySQL

## Our Idea
The goal of this project was to reduce manual hospital work and provide faster healthcare services using AI and automation. We wanted to create a platform that makes appointment booking, symptom analysis, and patient management easier and more efficient.

## Challenges Faced
- Integrating AI-based symptom prediction
- Managing frontend and backend connectivity
- Handling appointment conflicts
- Ensuring secure user authentication

## Solutions
- Implemented AI logic for symptom analysis
- Used Django for smooth backend integration
- Added validation to avoid duplicate bookings
- Used Django authentication for security
